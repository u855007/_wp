🐍 貪吃蛇遊戲

AI 輔助學習專案報告

報告人：林峻德

📋 專案概述

專案目標

學習 Node.js 後端開發
學習 HTML5 Canvas 遊戲開發
使用 AI 輔助程式開發工具

功能特色

🐍 經典貪吃蛇遊戲
📊 分數/等級/速度系統
🏆 排行榜功能
🎮 流暢鍵盤控制

🛠️ 技術架構

本專案使用以下技術：

HTML5 Canvas JavaScript ES6+ Node.js Express REST API

系統架構

玩家瀏覽器 → Node.js 後端 (Port 4000) → 遊戲 API / 排行榜

🤖 AI 輔助工具

本專案使用以下 AI 工具輔助開發：

ChatGPT Claude

ChatGPT 用途

• 程式碼生成
• 遊戲邏輯撰寫
• 除錯輔助

Claude 用途

• 程式碼審查
• 優化建議
• 程式碼分析

💡 AI 使用技巧

  • 清楚描述需求 - 指定技術棧、功能需求
  • 驗證程式碼 - 理解每行程式碼用途
  • 持續學習 - 比較 AI 方案與自己的想法
  • 有效提問 - 提供錯誤訊息請 AI 除錯

🎮 遊戲功能

  • 20x20 網格遊戲區域
  • 方向鍵 / WASD 控制移動
  • 空格鍵暫停/繼續
  • 吃到食物加分並變長
  • 每 50 分升一等級，速度加快
  • 撞牆或撞自己遊戲結束
  • 顯示分數、等級、速度

🔌 API 接口

遊戲排行榜 API

POST /api/game/submit - 提交分數
GET /api/game/leaderboard - 獲取排行榜

健康檢查

GET /api/health - 檢查伺服器狀態

📝 學習心得

  • 掌握 Node.js + Express 後端開發
  • 學會 HTML5 Canvas 遊戲開發
  • 了解 REST API 設計與實作
  • 學會使用 AI 輔助程式開發
  • 理解遊戲邏輯與碰撞檢測
