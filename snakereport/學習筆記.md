## FastAPI 學習筆記

### 1. 安裝

```bash
pip install fastapi uvicorn httpx
```

### 2. 基本結構

```python
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

class SearchRequest(BaseModel):
    query: str
```

### 3. 非同步處理

```python
@app.post("/search")
async def search(request: SearchRequest):
    # async await
    return {"results": [...]}
```

### 4. OpenAI API 串接

```python
import httpx
import os

async with httpx.AsyncClient() as client:
    response = await client.post(
        "https://api.openai.com/v1/chat/completions",
        headers={"Authorization": f"Bearer {api_key}"},
        json={"model": "gpt-4", "messages": [...]}
    )
```

### 1. Cargo 專案

```bash
cargo new search-engine-rust
```

### 2. Cargo.toml

```toml
[dependencies]
serde = { version = "1.0", features = ["derive"] }
tantivy = "0.22"
```

### 3. Tantivy 基本使用

```rust
use tantivy::{Index, IndexWriter, doc};

let index = Index::create_in_ram(schema);
let mut writer = index.writer(50_000_000)?;

writer.add_document(doc!(
    title_field => "Python",
    url_field => "https://python.org",
))?;
writer.commit()?;
```

### 4. 搜尋查詢

```rust
let query_parser = QueryParser::for_index(&index, fields);
let query = query_parser.parse_query("Python")?;
let top_docs = searcher.search(&query, &TopDocs::with_limit(10))?;
```

### 5. Ownership 概念

```rust
// 簡單例子
fn main() {
    let s1 = String::from("hello");
    let s2 = s1; // s1 被移動到 s2
    // println!("{}", s1); // 錯誤：s1 已移動
    println!("{}", s2); // 正確
}
```

## 踩坑記錄

### 1. Node.js CORS 問題

**問題**: 前端無法訪問後端 API

**解決**:
```javascript
const cors = require('cors');
app.use(cors());
```

### 2. Python httpx 異步

**問題**: 使用同步方式請求 API

**解決**:
```python
import httpx
async with httpx.AsyncClient() as client:
    await client.post(...)
```

### 3. Rust Cargo 編譯慢

**問題**: 首次編譯 Tantivy 很慢

**解決**:
```bash
cargo build --release
# 等待首次編譯完成後會快很多
```

### 4. OpenAI API Key 設定

**問題**: API Key 無效

**解決**:
```bash
# 在 .env 設定
OPENAI_API_KEY=sk-...
```
