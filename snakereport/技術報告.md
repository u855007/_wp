# 🐍 貪吃蛇遊戲 - AI 輔助學習專案報告

## 1. 專案概述

本專案為一個完整的 **AI 輔助貪吃蛇遊戲**，結合 Node.js 後端與 HTML5 Canvas 前端，旨在學習現代網頁開發技術與 AI 工具應用。

### 1.1 專案目標
- 學習 Node.js + Express 後端開發
- 掌握 HTML5 Canvas 遊戲開發
- 了解 AI 輔助程式開發工具之應用
- 實踐完整的專案開發流程

### 1.2 功能特色
- 🐍 經典貪吃蛇遊戲玩法
- 📊 分數、等級、速度系統
- 🏆 排行榜功能
- 🎮 流暢的鍵盤控制
- 🤖 AI 輔助程式開發

## 4. 遊戲實現說明

### 4.1 遊戲核心邏輯

```javascript
// 蛇的移動
direction = { x: 1, y: 0 };  // 向右移動

// 更新蛇的位置
head = { x: snake[0].x + direction.x, y: snake[0].y + direction.y };
snake.unshift(head);  // 添加新頭部
snake.pop();         // 移除尾部（不吃食物時）
```

### 4.2 遊戲功能

| 功能 | 說明 |
|-----|------|
| 移動控制 | 方向鍵、WASD、空格暫停 |
| 食物系統 | 隨機生成、檢查碰撞 |
| 等級系統 | 每 50 分升一等級 |
| 速度系統 | 等級越高，速度越快 |
| 遊戲結束 | 撞牆或撞自己 |

### 4.3 API 接口

**POST** `/api/game/submit` - 提交分數

```json
Request:
{
  "name": "Player",
  "score": 150,
  "level": 3
}

Response:
{
  "success": true,
  "rank": 1
}
```

**GET** `/api/game/leaderboard` - 獲取排行榜

```json
Response:
{
  "leaderboard": [
    { "name": "Player1", "score": 150, "level": 3, "date": "2024-01-15" }
  ]
}
```

## 5. 安裝與執行

### 5.1 前置需求
- Node.js 18+
- 現代瀏覽器（Chrome、Firefox、Edge）

### 5.2 安裝步驟

```bash
# 進入 Node.js 後端目錄
cd server-node

# 安裝依賴
npm install

# 啟動伺服器
npm run dev
```

### 5.3 執行遊戲

1. 打開瀏覽器訪問 `http://localhost:4000/snake-game/index.html`
2. 或直接打開 `snake-game/index.html` 檔案

### 5.4 控制方式

| 按鍵 | 功能 |
|-----|------|
| ↑ / W | 向上移動 |
| ↓ / S | 向下移動 |
| ← / A | 向左移動 |
| → / D | 向右移動 |
| 空格 | 暫停/繼續 |
